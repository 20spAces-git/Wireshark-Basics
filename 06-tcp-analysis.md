## Key Concepts

### What is TCP?

TCP (Transmission Control Protocol) is a connection-oriented protocol that provides reliable communication between systems.

TCP ensures:

- Data arrives correctly
- Data arrives in order
- Lost packets can be retransmitted

---

## The TCP Three-Way Handshake

TCP connections begin with a process known as the three-way handshake:

1. SYN
2. SYN-ACK
3. ACK

This process establishes communication between two devices before data transfer begins.
