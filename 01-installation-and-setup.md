# Lab 01 - Wireshark Installation and Setup

## Objective

Install Wireshark and configure the system for packet capture and network traffic analysis.

---

## Lab Environment

### Operating System

- Windows 11

### Software Installed

- Wireshark 4.6.6 (64-bit)
- Npcap 1.88

---

## Procedure

### Step 1 - Download Wireshark

Downloaded the Wireshark Windows x64 installer from the official Wireshark website.

### Step 2 - Launch Installer

Executed the installer and accepted the default installation options.

### Step 3 - Install Npcap

Installed Npcap 1.88 to enable packet capture functionality.

Npcap is required for Wireshark to capture live network traffic.

### Step 4 - Complete Installation

Finished installation and launched Wireshark successfully.

### Step 5 - Verify Interfaces

Opened Wireshark and verified that network interfaces were visible and available for packet capture.

## What I Learned

Wireshark is a network protocol analyzer that allows security professionals to capture and inspect network traffic.

Npcap provides the packet capture functionality required by Wireshark to monitor communications on a network.

Wireshark is commonly used by:

- Cybersecurity Analysts
- SOC Analysts
- DFIR Analysts
- Network Administrators
- Incident Responders

---

## Cybersecurity Relevance

Packet analysis is an important skill used to:

- Investigate network incidents
- Troubleshoot connectivity issues
- Detect suspicious activity
- Understand protocol behavior
- Support digital forensic investigations

---

## Safety Considerations

- Only capture traffic on systems and networks you own or are authorized to analyze.
- Follow organizational policies when performing packet captures.
- Avoid collecting sensitive information without authorization.
